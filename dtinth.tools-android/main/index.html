dtinth’s Tools

A personal Android enhancement app that I use on my device.

  • Morse code notifier vibrates my phone when I receive a notification.

  • Notification exfiltrator sends the received notifications into a remote server. It has end-to-end encryption, and aims to be resilient. The primary use case is to integrate with my personal chatbot, expense tracker, and home automator, “automatron”.

Sounds cool and want something like this? I encourage you to create your own personal Android enhancement app. Feel free to look at my code and reuse parts of it!
