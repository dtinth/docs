Authoring guide

Writing docs

To develop a documentation in individual project, first create docs/antora.yml:

name: project-name
version: main
title: 'Project name'

Then create modules/ROOT/pages/index.adoc with the contents.

Previewing docs

In VS Code, AsciiDoc files can be previewed using the AsciiDoc extension. This gives faster feedback but has lower fidelity. For high fidelity preview, install the development server:

npm install --global dtinth/docs

Then run this script from project root directory:

dtinth-docs-server

Creating a navigation file

Add this to docs/antora.yml:

nav:
- modules/ROOT/nav.adoc

Then create docs/modules/ROOT/nav.adoc:

* xref:index.adoc[]

Publishing

  1. Add a new entry to the docs.dt.in.th spreadsheet on Google Sheets.

  2. Rebuild the site on GitHub Actions
